Backbone Project

Recommendation Website Project

A website was build for academic project. User are able to review movies and get new movie recommendation based on user's preference. One Page structure ared used, where JavaScript Backbone framework were majorly used to finish all the interaction.

The recommendation algorithm is developed based content filtering which are based on movie description and profile of the user preference. The movies in final list are picked from high similarity rate of all movies. The rate is computed from Jacrood corfficient formula: rate = (Present_overlapped weight)/(Abscence Data_ total weight) = Sum(p_i*v_i)/SQRT(Sum(p_i^2)*Sum(v_i^2)),p_i:user history rate for movie i, v_i:movie i description; The data from the demo website is generated manually, in JSON.
